# Step 1: Create your Django project

* If you don't have `mkvirtualenv`

```bash
# usign pip
# (window)
pip install virtualenvwrapper-win
# (Linux & Mac)
pip install virtualenvwrapper-win
```
* virtual Environments

```bash
mkvirtualenv djreact
pip install Django
django-admin startproject djreact
mv djreact django
# (window)
rename djreact django
```
